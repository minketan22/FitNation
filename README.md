# 💪 TanuGym - Gym & Fitness Website

A modern, fully responsive gym and fitness website built with **React** and **Vite**. The project features a clean UI, smooth animations, reusable React components, and a mobile-first responsive design.

🔗 **Live Demo:**(https://fit-nation-kvfu.vercel.app/)

---

## ✨ Features

- 🎨 Modern and professional UI
- 📱 Fully responsive for Desktop, Tablet, and Mobile
- ⚡ Built with React + Vite
- 🧩 Reusable React components
- 🔄 Dynamic rendering using `map()`
- 🎬 Smooth scroll animations with AOS
- 📊 Statistics section
- 💪 Programs section
- 👨‍🏫 Trainers section
- 💰 Pricing plans
- ⭐ Testimonials
- 🖼️ Gallery
- 📞 Contact section
- 🔗 Smooth navigation
- 📌 Sticky navigation bar
- 🚀 Deployed on Vercel

## 🚀 Getting Started

### Clone the Repository

```bash
git clone YOUR_GITHUB_LINK
```

### Navigate to the Project

```bash
cd TanuGym
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
